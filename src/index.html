About Evan Rosa

Hi, I'm Evan Rosa. A Washington DC based digital marketer, front-end developer, and digital analytics professional. I'm very passionate about creating cohesive brand experiences and achieving transformational business results through digitally-driven creative marketing strategy. As a current digital analytics professional, I've had the unique opportunity to work with FDA, HHS, and non-profit websites to understand website usage and assisted in launching a variety of business programs. I define myself as a hybrid professional between the business of digital marketing and analytics as well as the technical side of front-end web development. I find that these skillsets are intertwined and are necessary to building a successful website or app. When I'm not working, I can usually be found jamming out on my guitar, traveling with my family or exploring the culinary world.

Evan Rosa's Specializations

In my current role I manage UA architectures and marketing tag strategies focusing on data accuracy and leveraging resources to understand website usage.   • Digital Marketing

Digital marketing is simply achieving marketing objectives through applying digital technologies, methodologies, and media. It starts at the discovery level and figuring out how to solve a problem. It places heavy importance on the type of content, titles, links, structured data markup, meta tags and keywords used so that search engines like Google can put your site in front of users looking for related content. Here you'll find my digital marketing skillset. I've used Screaming Frog, MailChimp, Adwords, Google Search Console, and CrazyEgg in a variety of projects. Feel free to click on any of the icons to learn more.

I've used Screaming Frog to audit a variety of websites. It's a great website crawler which spiders websites’ links, images, CSS, script and apps like an actual search engine to evaluate onsite SEO. It allows me to really dig deep and find any potential issues as well as help develop effective strategies.

Everyone in digital marketing should know what and how to use MailChimp. I've used this effective email marketing service to engage users with new content updates and more. It's a great tool for building a customer database, managing email contacts, creating and sending large scale email campaigns, and analyzing those campaigns for improvement.

Adwords is a service that lets you compete online to provide a brief advertisement to users based on relevant keywords for your business. It’s a bid-based system and it can most definitely increase traffic to your website. I'm Adwords certified and I use it with clients to attract new business and increase ROI.

Google Search Console is a go-to service for any digital marketer. One of its primary functions is to get a website ranked and get more traffic through SEO. It allows me to submit and check sitemaps, check and set crawl rates, submit a robot.txt file, and even view organic keywords. I use this tool to make decisions based off of real data instead of going in blindly.

If you're not familiar with CrazyEgg, it's a heatmap tool that takes digital snapshots that display your visitors' click behavior. It provides data rich reports that can provide insights into how users are interacting with a website to convert more visitors into revenue and goals.

  • Web Development

New and exciting technologies are always being introduced and knowing how to leverage those various platforms into a project can be difficult. With these technologies I am able to build websites, web apps, e-commerce tools, dashboards and much more. I am a big fan of building code that’s clean, organized, and semantic. I believe it’s best to build maintainable, efficient, and readable code to avoid hiccups or bugs in the future. The way I see it, it’s better to build something right the first time. Some technologies that I may use include:

  • HTML5
  • CSS3, SASS, SCSS
  • JavaScript
  • JQuery
  • Github
  • WordPress
  • Shopify
  • Jekyll
  • BootStrap
  • Foundation

HTML is to the web is what bricks are to a building. This is the code that describes and structures a webpage and allows for a website to be seen on the web. As the developer, I use it to format all text within a project such as titles, headings, images, and other links. All web related content uses HTML, so this language is essential for anything web related.

CSS like HTML is an essential tool in web development. Think of it as the design or style language of the project. It does everything from building specific element colors to creating responsive layouts. I use CSS along with SASS and SCSS, which are advanced styling code languages, to create the look and feel of a website or web-app.

JavaScript is the language that drives the web. It accounts for all of the functionality and interactivity on a website or app. It can be used to create pretty much anything on the web such as a scroll, e-commerce, or even an animation feature. It is also the foundation for major frameworks like Node.js, Angular, and React. I use JavaScript to develop unique interactive user experiences that don’t only look good but are useful for the end-user.

JQuery is a compliment to JavaScript that allows for interactive code to be written at a much more efficient rate. Rather than “re-inventing the wheel” and writing JavaScript from scratch, it provides JavaScript plugins and ready-made elements to assist in customizing any interactivity or functionality. With JQuery in my toolbox, the development time could be cut down significantly.

Bootstrap is a mobile and responsive framework that allows for fast development. It comes with pre-exiting HTML, CSS, and JavaScript elements and has a ton of benefits that give me the ability to create custom websites and web apps. Some of those benefits include a responsive grid system, custom components, JavaScript plugins, and consistency on all devices. As a Front-End Developer using Bootstrap, it has definitely saved me time and in-turn can save money.

Foundation is a front-end framework like Bootstrap. Its advanced features make it easy to design and prototype beautiful responsive websites that work on any device. Its framework is semantic, mobile ready, customizable and comes with tons of custom plugins. Not a fan of other frameworks or need more advanced features for your app or website? Foundation is the tool you need to build-out any project.

Wordpress, which powers about 25% of the web, is a content management system that supports the creation of content using a simple user interface. It runs on PHP and utilizes databases for data storage. So if you run an online store or blog that contains constant content updates then this is one of many platforms for you to consider. It’s easy to use, flexible, and contains tons of plugins for fast development. As a Front-end Developer, I use it to create projects that require ongoing content updates.

Jekyll is known as a simple, blog aware, static site generator. Unlike Wordpress, it’s very minimalistic and doesn’t need a back-end database or server-side code. It utilizes front-end files to build static websites from dynamic components such as templates, partials, liquid code, markdown, etc. What’s great about Jekyll is that it’s very fast, secure, easy to update, and completely customizable. If you're not a fan of Wordpress, then Jekyll is another option for creating blogs, e-commerce, or other dynamic sites with constant updates.

If you have an e-commerce site then Shopify might be the perfect solution. It's a content management system specifically for e-commerce and provides plenty of resources to get any e-commerce site up and running. I've used it to build a templated website for a friend and they love it. Check out his site Ayoba-Yo to see what I've done.

If you're any kind of developer then you probably have used Git or Github. It's a developement platform for a version control repository. Meaning that it can back-up a codebase so that so that I can keep track of the various changes that occur within the development process. I use this tool to back up code, solve issues by working with the Github community, collaborate with team members and much more.

  • Digital Analytics

Digital Analytics is becoming one of the most sought after skillsets in the web community. It allows for the measurement, collection, analysis and reporting of web data, such as a website, so that I can understand an audience and optimize an online business. I’ve been using analytical tools like Google Analytics, Google Tag Manager and others for over 7 years to help understand how to promote an online business and understand how users are using websites or apps. With this data I can not only find weaknesses within a website but also build and improve upon a web startegy.

Google Analytics is the go-to web analytics platform for many web marketers. It gives you the ability to collect useful non-invasive data to help understand your audience, their behavior and how you can leverage that data to build a better online product. I use Google Analytics not only to understand basic user interactions and behavior but also to create goals to increase conversions and to better track the events within a user’s web session. I have a Google Analytics Individual Qualification (GAIQ) certificate that gives anyone I work with piece of mind knowing that they know what and who they're working with.

Google Tag Manager is an amazing tool that can work with Google Analytics and other analytic and marketing platforms. It allows me to quickly create tags and other various elements on a website or app for the intended use of traffic analysis and marketing. With this tool, I can create AdWords, Google Analytics, Firebase, Floodlight and other custom tags to help you analyze your traffic on a whole new level.

Google Data Studio is a fairly new tool that allows me to create custom dashboards with a variety of datasets (not just Google Analytics). I can use this tool to collect data from a bunch of third part websites (i.e. Facebook, MailChimp, AdRoll, Salesforce etc) and pull that data into a easily digestable report. I can even go as far as creating calculated values to dig deeper and understand website usage.

SQL has been around for a while now but that doesn't mean it's not a powerful tool to utilize. I use it to query databases to learn more about website users. It's very flexible and allows me to tie data points together to build, understand, and achieve your goals.

Tableau is a great tool that can help any business understand thier data. It's feature rich and can connect to almost any database. I use it to create visualizations, analyze data, and connect the dots to understand dynamic relationships.

SAP Business Objects is a great tool that can help me view, sort, and analyze data. I can create ad-hoc queries and reports to help you develop a better business strategy.
